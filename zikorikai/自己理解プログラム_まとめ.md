# 0-2_プログラムのゴールを明確にする

## このプログラムに参加した理由

1. 自分が本当にやりたいことを見つけたいから
2. 過去からの実体験を元に、どんな事業が自分に合っているのか分からなかったから
3. 自分に自信がなくて、「本当にこれで大丈夫か」と自分の意見に確信が持てないから

## このプログラムが終わった後、どうなっていたいか

1. 自分が本当にやりたい事が見つかっている
2. 具体的にどんなジャンルが向いてるか分かる
3. 自分に自信が持てて、自分の決断に確信が持てる

## 最終的なゴール

誰もが自分の強みだけで生きていける「自立の仕組み」を完成させる

# 1-0_自己理解メソッドを学んでの気づき

## 「やりたいこと」を見つける重要性

- 自分の「好きなこと」「得意なこと」「価値観」が整理されていないと、何をしていいか分からなくなる
- やりたいことが明確になると、日々の選択に迷いがなくなる

## 自己理解の3つの要素

1. **好きなこと（興味）**: 自然と興味が湧くこと、時間を忘れて没頭できること
2. **得意なこと（才能）**: 無意識にできること、人よりも楽にできること
3. **大事なこと（価値観）**: 人生で大切にしたい状態

## 気づき

- 自分の強みを活かせる仕事を選ぶことが重要
- 価値観に沿った生き方をすることで、充実感が得られる
- 得意なことは「才能」であり、無意識にできること

# 2-1_モチベーショングラフの内容について質問に答える

## 1. ポジティブ体験の上位5つの共通点

1. **1人で没頭できる環境**: Web制作、独立など、1人で集中して取り組める状態
2. **目標が明確**: 具体的なゴールに向かって戦略を立てている時
3. **自分の強みが発揮できる**: デザイン、分析、戦略立案など
4. **自由度が高い**: 自分で意思決定ができる環境
5. **新しい発見・成長**: 留学、AI学習など、新しいことを学ぶ機会

## 2. ネガティブ体験の上位5つの共通点

1. **強制される環境**: 学校の勉強、集団スポーツなど
2. **目的が分からない**: なぜそれをするのか理解できない
3. **自由がない**: 長時間労働、監視される環境
4. **1人の時間がない**: 常に誰かと一緒にいなければならない
5. **評価される環境**: テストの点数、学歴で判断される

## 3. 人生で繰り返しているパターン

- **ポジティブ**: 新しいことを始める → 夢中になる → 成果が出る
- **ネガティブ**: 目的がないことを強制される → 興味がなくなる → 自己肯定感が下がる

# 5-0_自分・他人方向の長所・短所パターンまとめ

## 自分方向の長所パターン

1. **戦略やアイデアを練るために、1人で深く思考する時間や環境を作る**
2. **失敗しないように、成功事例を調査して分析し、真似する**
3. **目的を達成するために、計画を明確にする**
4. **複雑な情報を分析して、相手が判断しやすい情報にする**
5. **未来の理想像から逆算し、具体的な数値目標やタスクを設定する**
6. **フレームワークを使って情報を整理する**
7. **目的達成に足りない知識を学習する**
8. **過去の失敗を繰り返さないように振り返りをする**

## 他人方向の長所パターン

1. **相手の課題を把握するために、会議をする**
2. **複雑な情報を分析して、結論と根拠を明確にして提示する**
3. **競合を分析して、優位性のある企画を立案する**
4. **目的達成のために情報を複数人に伝える**
5. **トラブルが発生して解決するために、課題管理表を作成して1つずつ報告する**

## 自分方向の短所パターン

1. **目的がない中で結論を出す**
2. **変化のない単純作業・マニュアル業務は、すぐに飽きてしまい、辞めて、別の新しい事をする**
3. **1人で考える時間がない環境で、戦略・分析をする**
4. **じっくり考える時間がない中で決断する**

## 他人方向の短所パターン

1. **会話や相手の価値を「生産性」や「目的」で無意識に評価する**
2. **相手の感情を無視し、事実に基づいた意見をする**
3. **目的がない状態で、複数人との飲み会や初めての人と会う**
4. **集団での感情的な共感を求められると、関与を拒否する**

# 5-3_ストレングスファインダーで矛盾している資質を活かす3パターン

## 矛盾1: 未来志向 × 目標志向

### どう発揮される？
- 独立した直後に未来の3年後、5年後、10年後以降も漠然と考えていた
- 目の前の目標だけを考えても、自分の中でワクワクしない為、大きい未来を考えて、そこから逆引きをして目標を設定していた

### オリジナルな長所パターン
- **未来の理想像から逆算し、具体的な数値目標やタスクを設定する**

## 矛盾2: 目標志向 × 着想

### どう発揮される？
- 目標志向が明確になると、アイデアがたくさん出てきたり、新しい事を検証しようとする
- 目標志向のプロセスに沿った、新しいアイデアが出る

### オリジナルな長所パターン
- **目的が明確になると、達成手段のアイデアを収集する**

## 矛盾3: 目標志向 × 適応性

### どう発揮される？
- スケジュールの期日や目標が明確に決まっていても、外的・内的要因にて、変化が求められる場面や、緊急性もあるので、柔軟にその場の課題を明確にして、対応する

### オリジナルな長所パターン
- **目標を達成するために、チーム内で会議をする**
- **トラブルが発生して解決するために、課題管理表を作成して1つずつ報告する**
- **目標を達成するために、進捗管理をする**

# SF資質分類_ストレングスファインダー資質分類

## 資質の4分類

### 実行力の資質
目標に向かって実行する力
- **目標志向**: 明確なゴールに向かって進む

### 影響力の資質
周囲に影響を与える力
- **最上志向**: 平均ではなく卓越を求める
- **自我**: 自分の存在価値を認められたい
- **自己確信**: 自分の判断を信じる
- **社交性**: 新しい人と出会うことを楽しむ

### 人間関係構築力の資質
人との関係を築く力
- **適応性**: 変化に柔軟に対応する
- **親密性**: 深い人間関係を築く

### 戦略的思考力の資質
情報を分析し、最適な方法を考える力
- **分析思考**: データや事実に基づいて判断する
- **着想**: 新しいアイデアを生み出す
- **戦略性**: 複数の選択肢から最適解を見つける
- **未来志向**: 未来のビジョンを描く
- **内省**: 自分自身について深く考える
- **原点思考**: 過去から学び、本質を見抜く
- **収集心**: 情報を集めることが好き

## 自分の特徴

- 戦略的思考力の資質が多い（7/14）
- 1人で考える時間が必要
- 分析と戦略立案が得意
- 目標が明確になると力を発揮する
