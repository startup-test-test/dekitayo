# Phase 1: 被リンク獲得戦略

## 1-1. プレスリリース転載 → 被リンク獲得

### 仕組み

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  Step 1: プレスリリースを見つける                            │
│    ・PR TIMES「プログラミング教室」で検索                    │
│    ・新規開校、キャンペーン、イベント情報を収集              │
│                                                              │
│  Step 2: 記事化して掲載                                      │
│    ・プレスリリースを元に記事作成                            │
│    ・教室の魅力を紹介                                        │
│    ・「無料掲載です」と明記                                  │
│                                                              │
│  Step 3: 教室に連絡                                          │
│    ・「貴社の教室を紹介しました」                            │
│    ・「よろしければリンクを貼っていただけませんか」          │
│                                                              │
│  Step 4: 被リンク獲得                                        │
│    ・教室のHPから「掲載メディア」としてリンク               │
│    ・ドメインパワー上昇                                      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 具体的な手順

#### Step 1: プレスリリースを探す

**PR TIMESで検索:**
- https://prtimes.jp/
- 検索キーワード:
  - 「プログラミング教室」
  - 「プログラミングスクール 子供」
  - 「小学生 プログラミング」
  - 「ロボット教室」
  - 「Scratch 教室」

**狙い目のプレスリリース:**
- 新規開校のお知らせ
- 新コース開設
- キャンペーン情報
- イベント開催
- 受賞・認定のお知らせ

#### Step 2: 記事を作成

**記事テンプレート:**

```markdown

# Phase 2: 教育心理学コンテンツ

## 2-1. 学ぶべき分野

### 基礎知識

| 分野 | 内容 | おすすめ書籍/リソース |
|------|------|---------------------|
| 自己肯定感 | 定義、測定方法、高め方 | 「自己肯定感の教科書」中島輝 |
| 発達心理学 | 子供の発達段階 | 「子どもの発達心理学」 |
| 動機づけ理論 | 内発的動機づけ、自己決定理論 | 「モチベーション3.0」ダニエル・ピンク |
| 成長マインドセット | 固定 vs 成長マインドセット | 「MINDSET」キャロル・ドゥエック |
| 褒め方の心理学 | 効果的な褒め方、逆効果な褒め方 | 各種論文 |

### 学習ロードマップ

```
Month 1-2: 基礎理論
  ├── 自己肯定感とは何か
  ├── 発達段階の理解
  └── 基本書籍を3冊読む

Month 3-4: 応用
  ├── プログラミング教育との接続
  ├── 「なぜプログラミングが自己肯定感を高めるか」の理論構築
  └── 記事執筆開始

Month 5-6: 体系化
  ├── 独自メソッドの構築
  ├── 保護者向けガイドの作成
  └── E-E-A-Tの確立
```

```

    ↓

    ↓

    ↓
