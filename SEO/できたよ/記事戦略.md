# 記事の型（テンプレート）

## 悩み共感型の記事構成

```
1. 導入（共感）
   「〇〇で悩んでいませんか？」
   └ 親の悩みを代弁、感情に寄り添う

2. 問題提起
   「実は、それは〇〇のサインかもしれません」
   └ 悩みを言語化、チェックリストなど

3. 原因解説
   「なぜそうなるのか」
   └ 心理学・教育学のエビデンス

4. 解決の方向性
   「でも、大丈夫です。〇〇で変われます」
   └ 希望を提示

5. 次のステップ
   「まずは〇〇から始めてみませんか？」
   └ 関連記事へ誘導
```

## プログラミング接続型の記事構成

```
1. 導入
   「〇〇な子供には、プログラミングが向いています」
   └ 結論ファースト

2. 理由1〜5
   「なぜなら〇〇だからです」
   └ 論理的に説明、エビデンス

3. 実例
   「実際に〇〇な子供が変わった事例」
   └ 体験談、ストーリー

4. 注意点
   「ただし、〇〇には注意が必要です」
   └ 信頼性を高める

5. 行動喚起
   「まずは体験から始めてみませんか？」
   └ 教室紹介ページへ誘導
```
