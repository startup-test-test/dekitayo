# 弱者のSEO戦略 - コエテコに勝つための設計書

## 前提条件

| 項目 | あなた | コエテコ |
|------|--------|----------|
| ドメインパワー | 0 | 高（運営年数長い） |
| 記事数 | 0 | 1,000+ |
| 地域カバー | 0 | 198地域で1位 |
| 掲載教室数 | 0 | 13,000+ |
| リソース | 限定的 | 豊富 |

**結論: 正面衝突では100%負ける。別の土俵で戦う。**

---

## コエテコの弱点分析

### 1. CVに遠いトラフィックが83%

```
コエテコの流入構造:

CapsLock解除で来た人 → プログラミング教室に興味ある？ → ほぼNO
画像検索で来た人    → プログラミング教室に興味ある？ → ほぼNO
Twitter検索で来た人 → プログラミング教室に興味ある？ → ほぼNO
```

**弱点**: 大量のトラフィックを集めているが、CVRは極めて低いはず

### 2. 「広く浅く」の限界

| カテゴリ | コエテコの状況 |
|---------|--------------|
| PC操作・トラブル | 106記事（専門サイトではない） |
| SNS活用 | 59記事（専門サイトではない） |
| 英語学習 | 104記事（専門サイトではない） |

**弱点**: どのカテゴリも「専門サイト」ではない。E-E-A-Tが弱い。

### 3. 情報の深さが浅い

- 1記事あたりの専門性は高くない
- 「とりあえず記事を量産」している印象
- ユーザーの本当の悩みに深く答えていない可能性

### 4. ターゲットが曖昧

```
コエテコのターゲット:
├─ 小学生の保護者 ← メイン
├─ 中高生の保護者
├─ 社会人（転職）
├─ 一般ユーザー ← 67.7%（誰？）
└─ 初心者
```

**弱点**: 「誰に向けて書いているか」が不明確な記事が多い


## 戦略1: 超ニッチ特化型（推奨度: ★★★）

### コンセプト
**「小学生 × Scratch × 特定の悩み」に絞り込む**

コエテコが「プログラミング全般」を広くカバーしているなら、
あなたは「Scratchで挫折した子の親」だけに刺さるサイトを作る。

### 具体的なニッチ案

| ニッチ | 検索例 | コエテコの対応 |
|--------|--------|---------------|
| Scratch挫折復帰 | 「Scratch 子供 飽きた」「Scratch 続かない」 | 薄い |
| 発達特性×プログラミング | 「ADHD プログラミング」「ASD 習い事」 | ほぼない |
| 不登校×プログラミング | 「不登校 オンライン 習い事」 | ほぼない |
| 女の子向け | 「女の子 プログラミング」「プログラミング 女子」 | 薄い |
| 地方特化 | 「〇〇県 プログラミング教室 少ない」 | 地域ページはあるが深くない |

### 実行プラン

```
Phase 1: 1つのニッチで30記事（3ヶ月）
         ↓
Phase 2: そのニッチで検索1位を10個獲得
         ↓
Phase 3: 隣接ニッチに横展開
```

### なぜ勝てるか

```
「Scratch 子供 続かない」で検索した親が求めるもの:

コエテコの記事:
→ Scratchの始め方、教室一覧... （求めてない）

あなたの記事:
→ 続かない5つの原因と、うちの子が復活した方法
→ 実際に試した教材3つのレビュー
→ 「飽きた」と言われた時の声かけ集
→ 同じ悩みの親100人に聞いたアンケート結果

どちらを読みたいか？ → 明らか
```

## 戦略2: CV直結キーワード狙い撃ち（推奨度: ★★★）

### コンセプト
**情報収集キーワードを捨てて、購買直前キーワードだけ狙う**

コエテコは「広く集めて導く」戦略。
あなたは「買う気満々の人だけ集める」戦略。

### 狙うべきキーワードパターン

| パターン | 例 | 検索意図 |
|---------|-----|---------|
| 「A vs B」比較 | 「〇〇 △△ 比較」「〇〇 △△ どっち」 | 2択で迷っている |
| 料金・月謝 | 「〇〇教室 料金」「〇〇 月謝 高い」 | 申込直前 |
| 口コミ・評判 | 「〇〇 評判 悪い」「〇〇 やめた理由」 | 最終確認 |
| 体験談 | 「〇〇 体験 感想」「〇〇 通ってみた」 | リアルな声を求めている |
| 解約・退会 | 「〇〇 解約 方法」「〇〇 やめ方」 | 不満を持っている（乗り換え候補） |

### 具体的なキーワード例

```
【比較系】
・QUREO vs Scratch（どっちがいい）
・リタリコワンダー Tech Kids 比較
・オンライン vs 通学 プログラミング

【料金系】
・プログラミング教室 月謝 相場
・〇〇教室 料金 高い 理由
・プログラミング 習い事 費用対効果

【ネガティブ口コミ系】★狙い目
・〇〇教室 やめた
・〇〇教室 合わなかった
・プログラミング教室 意味ない

【体験談系】
・〇〇教室 〇ヶ月通った感想
・子供 プログラミング 効果 あった
```

### なぜこれが狙い目か

```
コエテコの立場:
- 教室を掲載してもらう立場
- ネガティブな記事は書きにくい
- 「〇〇教室 やめた理由」は書けない

あなたの立場:
- 中立的な立場で書ける
- ネガティブ情報も正直に書ける
- ユーザーの信頼を得やすい
```

## 戦略3: 1地域完全制圧（推奨度: ★★☆）

### コンセプト
**198地域で戦わず、1地域で圧倒的に勝つ**

コエテコは全国展開で「広く浅く」。
あなたは1地域で「狭く深く」。

### 実行プラン

```
【例: 横浜市に特化】

コエテコ: 「横浜 プログラミング教室」で1ページ
あなた:
├─ 横浜市全体のプログラミング教室完全ガイド
├─ 横浜市青葉区のプログラミング教室5選
├─ 横浜市港北区のプログラミング教室5選
├─ 横浜駅から通えるプログラミング教室
├─ 横浜市で体験無料のプログラミング教室
├─ 横浜市の安いプログラミング教室ランキング
├─ 横浜市で土日OKのプログラミング教室
├─ 【体験レポ】〇〇教室（横浜）に行ってきた
└─ ...（20記事以上）
```

### 選ぶべき地域の条件

| 条件 | 理由 |
|------|------|
| 人口30万人以上 | 検索ボリュームが必要 |
| 教室数が多すぎない | 記事にしやすい |
| 競合が弱い | 地方都市が狙い目 |
| 自分が詳しい | リアルな情報を書ける |

### おすすめ地域

```
狙い目（検索ボリュームあり & 競合弱め）:
・さいたま市
・千葉市
・川崎市
・相模原市
・堺市
・北九州市
・熊本市

```

## 戦略5: YouTube/SNS連携でE-E-A-T構築（推奨度: ★☆☆）

### コンセプト
**ブログだけでなく、顔出しでE-E-A-Tを構築**

### 実行プラン

```
YouTube:
├─ 「うちの子のScratch作品紹介」シリーズ
├─ 「プログラミング教室の体験レポ」
├─ 「〇〇教室に通って3ヶ月の変化」
└─ 顔出し + 実名でGoogleに「この人は本物」と認識させる

Blog:
├─ YouTube動画を埋め込み
├─ 動画の内容を記事化
└─ 相互リンクでSEO強化
```

### なぜ有効か

```
Googleの評価:
・顔出し、実名 = 信頼性が高い
・YouTubeでも活動 = 本物の専門家
・SNSフォロワーがいる = 影響力がある

コエテコは「企業サイト」
あなたは「リアルな保護者の声」
→ E-E-A-Tの「Experience」で勝てる
```
