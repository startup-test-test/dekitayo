## 2-4. コエテコの弱点

| 弱点 | 詳細 |
|------|------|
| **CVに遠いトラフィック83%** | CapsLock解除で来た人はプログラミング教室に興味ない |

| **ターゲットが曖昧** | 67.7%が「一般」=誰に向けて書いているか不明確 |

## 2-5. 狙い目キーワード分析

### コエテコが弱いキーワード（41件抽出）

| 条件 | 内容 |
|------|------|
| 条件1 | コエテコが4位以降（1位を取れていない） |
| 条件2 | 難易度40以下 |
| 条件3 | 流入500以上 |

### 結論

```
コエテコが「弱い」キーワード41件のうち、
プログラミング関連はたった3件。

→ コエテコはプログラミング教育では非常に強い
→ SNS、画像検索、副業などは手薄（でもCVに遠い）
→ 「弱いところを奪う」より「別の土俵で戦う」が正解
```

# 4. あなたの差別化ポイント

## 4-1. コエテコとの違い

| 項目 | コエテコ | あなた |
|------|---------|--------|
| **コンセプト** | 教室を探す | 子供の自己肯定感を高める |
| **専門性** | プログラミング教室一覧 | 教育心理学 × 自己肯定感 |
| **ターゲット** | 「教室を探している人」 | 「子供に自信をつけさせたい親」 |
| **掲載モデル** | 有料掲載 | **無料掲載** |
| **被リンク** | 教室からのリンクなし | **教室からリンク獲得** |
| **記事の深さ** | 広く浅く | 狭く深く |

## 4-2. 勝てる理由

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  1. 無料掲載 → 教室が断る理由がない                         │
│                                                              │
│  2. 被リンク獲得 → ドメインパワーが自然に上がる             │
│                                                              │
│  3. 教育心理学 × 自己肯定感 → コエテコにない専門性          │
│                                                              │
│  4. 「教室を探す」ではなく「子供を救う」→ 情緒的価値        │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 4-3. コエテコが真似できない理由

- **専門性**: 教育心理学の知識が必要
- **ビジネスモデル**: 有料掲載で稼いでいるので無料掲載できない
- **コンテンツの深さ**: 1000記事を広く浅く作る戦略と相反
- **ターゲット**: 「教室を探す人」に最適化されたサイト設計

# 5. 被リンク獲得戦略

## 5-1. 戦略の全体像

```
無料掲載
  ↓
全国のプログラミング教室が登録
  ↓
各教室が自分のサイトからリンクを貼る
「当教室は〇〇に掲載されています」
  ↓
被リンク大量獲得
  ↓
ドメインパワー急上昇
  ↓
SEOで勝てる
```

## 5-2. プレスリリース転載 → 被リンク獲得

### 手順

```
Step 1: PR TIMESでプレスリリースを探す
        https://prtimes.jp/
        検索: 「プログラミング教室」「小学生 プログラミング」

Step 2: 記事にして掲載（連絡なしで先に掲載）

Step 3: サイトに実績ができたら教室に連絡

Step 4: 被リンク獲得
```

### 狙い目のプレスリリース

- 新規開校のお知らせ
- 新コース開設
- キャンペーン情報
- イベント開催
- 受賞・認定のお知らせ

### 連絡メールテンプレート

```
件名: 【掲載のご連絡】貴社プログラミング教室を紹介させていただきました

〇〇教室 ご担当者様

突然のご連絡失礼いたします。
「[サイト名]」を運営しております[名前]と申します。

当サイトは、教育心理学の観点から子供の自己肯定感を高める
プログラミング教育を紹介するメディアです。

この度、貴社のプレスリリースを拝見し、
素晴らしい取り組みだと感じ、当サイトで紹介させていただきました。

▼掲載記事
[記事URL]

※掲載は完全無料です
※内容の修正・削除のご要望があればお知らせください

もしよろしければ、貴社サイトの「メディア掲載」等で
当サイトをご紹介いただけますと大変嬉しく存じます。

今後も貴社の活動を応援しております。
何卒よろしくお願いいたします。

[署名]
```

### 期待効果

| 項目 | 数値 |
|------|------|
| 被リンク獲得率 | 20-30% |
| 月間連絡数 | 20件 |
| 月間被リンク | 4-6件 |
| 6ヶ月後 | 24-36件の被リンク |

## 5-3. 「鶏と卵」問題の解決

```
問題:
記事がないと掲載されない → 掲載されないと被リンクがない

解決策:
最初の10教室は「連絡なし」で掲載
  ↓
サイトに「実績」ができる（10教室掲載中）
  ↓
その後、新しい教室に連絡（信頼感UP）
  ↓
最初の10教室にも後から連絡
```

# 8. 無料掲載と有料プランの線引き

## 8-1. 基本戦略

```
Phase 1（Year 1）: 全て無料
  → 被リンク獲得が最優先
  → 教室数を増やすことが目的

Phase 2（Year 2〜）: 無料 + 有料オプション
  → 基本掲載は無料のまま
  → 付加価値に課金
```

| 公式サイトへのリンク | ○ | ○ |
| 写真掲載 | 1枚 | 10枚まで |
| 教室紹介文 | 200文字 | 2000文字 |
| **特集記事** | × | ○（月1回） |
| **トップページ掲載** | × | ○ |
| **地域ページ上位表示** | × | ○ |
| **口コミ返信機能** | × | ○ |
| **アクセス解析** | × | ○ |
| **バナー広告枠** | × | ○ |

## 10-2. 現実的なシナリオ

```

  → 月75万円

```

### パターンB: 無料掲載 → 有料プランへ移行

```
Year 1: 無料掲載で500教室獲得（被リンク獲得）
  ↓
Year 2: 有料プラン開始
  「上位表示」「特集記事」など付加価値
  ↓
有料プラン: 月1万円 × 100教室 = 月100万円
```

### パターンC: 複合型（推奨）

```
【収益内訳】
・アフィリエイト: 150件 × 5,000円 = 75万円
・有料プラン: 50教室 × 5,000円 = 25万円
・合計: 100万円/月

【コスト内訳】
・業務委託: 20万円
・サーバー・ツール: 2万円
・合計: 22万円/月

【利益】
・100万円 - 22万円 = 78万円/月
```
