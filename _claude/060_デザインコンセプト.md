# デザインコンセプト

## コンセプト

**「達成感」— 子供の「できた！」の喜びの瞬間を表現する**

```
ビジョン：「一人ひとりの『ものさし』で、生きられる世界」
    ↓
サイトの目的：子供の自己肯定感を高める
    ↓
デザインで表現：「できた！」という成功体験・達成感
```

---

## キーワード

- **達成感**：「できた！」の喜びの瞬間
- **温かみ**：親しみやすく、安心感がある
- **信頼感**：専門性があり、信頼できる
- **前向き**：明るく、希望がある
- **成長**：子供の可能性が広がるイメージ

## ロゴ

### サイト名

**「できたよ！」**（「！」あり）

理由：
- 「！」があることで喜び・達成感が伝わる
- 子供が嬉しそうに報告している感じが出る
- 印象に残りやすい

### コンセプト

**「できた！」の喜びの瞬間を表現**

```
イメージ案：
├── 「！」を星や光で表現
├── 手を上げて喜ぶシルエット
├── チェックマーク + 笑顔
└── 上向きの要素（成長・達成）
```

### カラー

- メイン：オレンジ（#FF6B35）
- 「！」やアクセント：イエロー（#FFD93D）

### バリエーション

| 種類 | 用途 | サイズ目安 |
|------|------|-----------|
| メインロゴ | ヘッダー、印刷物 | 横幅200px〜 |
| ファビコン用 | ブラウザタブ | 32×32px |
| SNS用 | Twitter、OGP | 正方形 1200×1200px |
| モノクロ | 背景が複雑な場合 | - |

## トーン＆マナー

### 目指す印象

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   温かみがあるけど、信頼できる                           │
│   親しみやすいけど、専門的                              │
│   明るいけど、落ち着いている                            │
│                                                         │
│   → 「子供向け」ではなく「親向け」のデザイン            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 言葉遣い

- 「〜しましょう」「〜できます」（前向き）
- 専門用語は使うが、必ず説明を添える
- 押し付けがましくない

### 写真・イラスト

- 子供の笑顔、達成した瞬間
- 明るい自然光のイメージ
- リアルな写真 > イラスト（信頼感）

## 禁止事項

- **子どもっぽすぎるデザイン**：信頼性が下がる（ターゲットは親）
- **暗い・重い印象**：コンセプトと矛盾
- **複雑すぎるレイアウト**：情報が伝わらない
- **原色の多用**：安っぽくなる
- **過度な装飾**：読みにくくなる
