## アーキテクチャ

```
┌─────────────────────────────────────────────────────────┐
│  ヘッドレスCMS構成                                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  WordPress（Xserver）                                   │
│  └── https://cms.dekitayo.jp                            │

```

### ヘッドレスCMSのメリット

| メリット | 説明 |
|---------|------|
| 表示速度 | フロントが静的生成で高速 |
| セキュリティ | WordPress管理画面が公開されない |
| 自由度 | デザイン・UIの自由度が高い |
| SEO | Core Web Vitals対策しやすい |

## バックエンド（CMS）

| 項目 | 選定 |
|------|------|
| CMS | WordPress（ヘッドレス） |
| サーバー | Xserver |
| 役割 | コンテンツ管理、API提供 |

### WordPressプラグイン

| プラグイン | 用途 |
|-----------|------|
| WPGraphQL | GraphQL API提供 |
| Advanced Custom Fields | カスタムフィールド |
| Yoast SEO | SEOメタデータ管理 |こ

---

## フロントエンド

| 項目 | 選定 |
|------|------|
| フレームワーク | Next.js |
| ホスティング | Vercel |
| 言語 | TypeScript |

### Next.jsを選んだ理由

| 理由 | 説明 |
|------|------|
| SEOに強い | サーバーサイドレンダリング対応 |
| 情報が多い | 困った時に調べやすい |
| Vercelと相性◎ | デプロイが簡単、無料枠あり |
| WordPress連携実績多い | ヘッドレスCMSの定番構成 |

## 禁止事項

| 禁止技術 | 理由 |
|---------|------|
| コインハイブ等のマイニング | 不正指令電磁的記録供用罪 |
| 無断トラッキング | 個人情報保護法違反 |
| 無断スクレイピング | 著作権・利用規約違反 |
| 不正アクセス | 不正アクセス禁止法違反 |

---

## 法令遵守

| 法令 | 対応 |
|------|------|
| 個人情報保護法 | プライバシーポリシー設置、Cookie同意バナー |
| 特定商取引法 | 必要に応じて表記ページ設置 |
| 著作権法 | 引用ルール遵守、無断転載禁止 |
| 景品表示法 | 誇大広告の禁止、PR表記 |
| 電気通信事業法 | 外部送信規律への対応（2023年施行） |
