# プレスリリース手順

## 概要

プレスリリースの作成から掲載までの手順

---

## 掲載サービス

| サービス | 料金 | 特徴 |
|---------|------|------|
| PR TIMES | 3万円〜 | 最大手、メディア転載多い |
| @Press | 3万円〜 | 配信先が多い |
| valuepress | 無料〜 | 無料プランあり |
| PRTIMES STORY | 無料 | ストーリー形式 |

---

## 手順

### Step 1：プレスリリース作成

```
1. タイトル作成（30文字以内）
2. リード文作成（概要）
3. 本文作成
4. 画像準備（ロゴ、サービス画像）
5. 会社概要作成
```

### Step 2：配信サービスに登録

```
1. アカウント作成
2. 企業情報登録
3. プレスリリース入稿
4. 画像アップロード
5. プレビュー確認
```

### Step 3：配信

```
1. 配信日時設定
2. 配信実行
3. 掲載確認
```

### Step 4：自社サイトに転載

```
1. /media/ ページに記事作成
2. プレスリリース内容を転載
3. 配信元へのリンク設置
4. 公開
```

---

## プレスリリースのタイミング

| タイミング | 内容 |
|-----------|------|
| サイト公開時 | サービス開始のお知らせ |
| 掲載教室数達成時 | 「掲載100教室突破」など |
| 新機能追加時 | 機能アップデート |
| 調査レポート発表時 | 独自調査の発表 |

---

## テンプレート

### タイトル例

```
子どもの自己肯定感を育むプログラミング教室ガイド「できたよ！」を公開
```

### リード文例

```
〇〇（運営者/会社名）は、子ども向けプログラミング教室の情報サイト
「できたよ！」を〇月〇日に公開いたしました。
```

### 本文構成

```
1. サービス概要
2. 開発背景・想い
3. 特徴（3つ程度）
4. 今後の展開
5. 会社概要
```

---

## 注意事項

- 誇大表現は避ける（景品表示法）
- 事実に基づいた内容のみ
- 画像の著作権に注意
