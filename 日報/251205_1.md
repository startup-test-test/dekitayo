## 1. デザインコンセプト確定

### カラーパレット

| 色 | コード | 用途 |
|-----|--------|------|
| Primary（オレンジ） | #FF6B35 | ボタン、アクセント |
| Secondary（ネイビー） | #1E3A5F | テキスト、見出し |
| Accent（イエロー） | #FFD93D | 強調 |
| Background | #FEFEFE | 背景 |
| Surface | #F5F5F5 | セクション背景 |

### デザイン方針

- ヘッダー・フッター：**白背景**（軽やかな印象）
- CTAセクション：**ネイビー背景**（教室掲載案内を目立たせる）
- オレンジ：温かみ、達成感（子ども向け）
- ネイビー：信頼感、知性（保護者向け）

## 4. ロゴデザイン

### 採用したロゴ

- 左側：ブランコに乗った男の子のキャラクター
- 右側：「できたよ！」のカラフルな文字
  - 「で」ピンク
  - 「き」水色
  - 「た」青
  - 「よ」緑
  - 「！」オレンジ

### 特徴

- 子ども向けで親しみやすい
- カラフルで楽しい雰囲気
- キャラクターが達成感・元気さを表現
- マスコットとして他ページでも活用可能

## 7. CMS連携

- 「ニュース」セクションでCMS記事を表示
- `getPosts()` で最新3件を取得
- 記事詳細ページ（`/posts/[slug]`）も作成済み

### GraphQL クエリ

```graphql
query GetPosts {
  posts(first: 10) {
    nodes {
      id
      title
      slug
      date
      excerpt
    }
  }
}
```

## 8. 技術メモ

### Tailwind CSS v4 の注意点

- `text-[var(--primary)]` が効かない場合あり
- → `style={{ color: "#FF6B35" }}` で直接指定

### Next.js 開発インジケーター

```ts
// next.config.ts
const nextConfig: NextConfig = {
  devIndicators: false,
};
```
