## 13. 根本的な課題の発見

### 問題点

```
PR TIMESの転載だけ
    ↓
教室にとっての独自価値がない
    ↓
「無料掲載しませんか？」と言っても響かない
```

### 教室が掲載したくなる理由が必要

1. トラフィック → まだない
2. SEO効果（被リンク） → ドメインパワーがまだ弱い
3. ターゲット層へのリーチ → まだ読者がいない

---

## 14. 解決策の方向性

### 3つのアプローチを並行

```
【土台】A: オリジナル記事で読者を集める
├── 教育コラム（自己肯定感、非認知能力）
├── 書籍レビュー
└── 教室選びガイド
    ↓ これがないと始まらない

【並行】B: 取材記事
└── 教室を取材して紹介 → 被リンク依頼

【並行】C: 教室リスト作成（Airbnb方式）
└── Googleで調べて先に掲載 → 後から連絡
```

## 18. 重要な気づき：PR TIMES登録の壁

### PR TIMES メディア登録の要件

```
必須項目：
├── 媒体URL（最低2つ必要）
├── 報道実績の証明（記事URLまたは画像）
└── 報道関係者であること（審査あり）
```

### 鶏と卵問題

```
PR TIMES登録したい → 記事が必要 → 記事がない →
プレスリリース転載できない → 記事が書けない → ループ...
```

### 結論：プレスリリースに頼らない

**Phase 1 のゴールを再定義：**

```
メディアサイトとして機能させる（プレスリリースなし）

コンテンツ（PR TIMES不要で書けるもの）：
├── 書籍レビュー（自分で読んだ本）
├── 教育コラム（自己肯定感、非認知能力）
├── 教室ガイド（選び方、比較）
└── 教室紹介（公式サイトから情報収集）
```
