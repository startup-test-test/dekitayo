# 日報 251201_1

## 本日の活動内容

### コエテコSEO分析

コエテコ（coeteco.jp）の競合分析を実施。

#### キーワード分析結果

| カテゴリ | 割合 |
|---------|------|
| How-to系（「プログラミングとは」等） | 93.6% |
| プログラミングスクール系 | 6.4% |
| エリア系 | **0.2%（3キーワードのみ）** |

#### 重要な発見

1. **コエテコはエリアSEOをほぼやっていない**
   - トップ1000キーワード中、エリアキーワードは3件のみ
   - 「エリア × プログラミング教室」の組み合わせはゼロ

2. **理由の分析**
   - 市場がまだ黎明期（比較検索が少ない）
   - 選択肢が少ないと人は検索しない
   - Google Mapsで代替される傾向
   - 口コミ・紹介経由が多い

## 学び・気づき

### SEOの法則：市場成熟度と検索行動

```
黎明期:   「〇〇とは」「〇〇 メリット」     ← 今のプログラミング教室
成長期:   「〇〇 比較」「〇〇 おすすめ」
成熟期:   「エリア × 〇〇」「〇〇 安い」   ← 脱毛、美容院、飲食店
飽和期:   「ブランド名 予約」「ブランド名 口コミ」
```

**選択肢が少ない業界 = 比較検索されない**

### 弱者の戦略

コエテコと同じ記事SEOで戦うのは無理（資金力勝負になる）。

#### 個人が勝てる戦略

1. **SNS × ニッチ特化**
   - 「人」が見える発信
   - 特定の悩みに深く寄り添う（学校が苦手な子など）
   - コミュニティを作る

2. **プレスリリース活用** ← 本日の重要アイデア
   - PR TIMESで「プログラミング教室」を検索
   - プレスリリースを出している = 宣伝意欲が高い
   - 「無料で掲載しませんか？」と営業
   - 断られにくい（無料 × 宣伝したい）

---

## 戦略の方向性

### 当初の想定
```
無料掲載 → 教室が集まる → 被リンク獲得 → ドメイン強化
```

### 課題
```
無料でも「見られていないサイト」には掲載してもらえない
```

### 解決策

**順番を変える：**

1. SNSで保護者コミュニティを作る
2. 「この層にリーチできます」と言える状態を作る
3. プレスリリース出稿教室に営業
4. 教室掲載が増える
5. コンテンツ充実 → SEO効果

## メモ

コエテコに有料掲載している教室 = 宣伝意欲がある = 無料なら掲載してくれる可能性高い

「無料掲載」だけでは弱い。掲載メリットを作る：
- SNSで紹介
- 記事を書く
